Short-term Vision

The short term vision is to develop some skills for web dev as I progress through the my dApp course material
The full application of these skills will be in a website for a charity stake pools called AfricaPools.io
The Cardano stake pool will also need to be developed and audited. The process of an audit will need to be stored in order to fuel the 2nd Stage
The stake pool itself will have 3 objectives

  1. Create a node in South Africa that will boost Cardano TX's and decentralization
  2. Use an ISPO model to fund itself and future stake pool strategies, as described in the Roadmap
  3. Use stake rewards to fuel local adoption of Cardano and donate to charity projects
